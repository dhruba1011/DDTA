Pointers in C

A pointer in C is a variable that stores the address of another variable. Here's an example:

int main() {
    int x = 10;
    int *p = &x;  // pointer to x
    printf("Value of x: %d", *p);  // Dereferencing the pointer
    return 0;
}

Memory Management

In C, we can dynamically allocate memory using malloc(), calloc(), and free it with free().
