Factorial Calculation

#include 

int main() {
    int n, i, factorial = 1;
    printf("Enter a number: ");
    scanf("%d", &n);
    
    for(i = 1; i <= n; i++) {
        factorial *= i;
    }
    printf("Factorial of %d is %d\n", n, factorial);
    return 0;
}

This program calculates the factorial of a given number.
